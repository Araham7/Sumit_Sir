## 📄 PAGE 14-20: TECHNICAL INFORMATION (Selection Guide) ⭐ (Very Important!)

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  🔥 KEY SYMBOLS & DEFINITIONS:                             │
│                                                             │
│  ✅ i = Gear Ratio                                         │
│  ✅ Mₙ₂ = Rated Output Torque (Nm)                         │
│  ✅ Mₐ₂ = Max Acceleration Output Torque (Nm)              │
│  ✅ Mₚ₂ = Emergency Stop Torque (Nm)                       │
│  ✅ n₁ = Input Speed (min⁻¹)                               │
│  ✅ n₂ = Output Speed (min⁻¹)                              │
│  ✅ φS = Standard Backlash (arcmin)                        │
│  ✅ φR = Reduced Backlash (arcmin)                         │
│  ✅ Cₜ = Torsional Stiffness (Nm/arcmin)                   │
│  ✅ R₂ max = Max Radial Load (N)                           │
│  ✅ A₂ max = Max Axial Load (N)                            │
│  ✅ η = Efficiency (%)                                     │
│  ✅ Jg = Mass Moment of Inertia (kgcm²)                    │
│                                                             │
│  🔥 SELECTION FORMULAS:                                    │
│                                                             │
│  ✅ i = n₁ / n₂                                            │
│                                                             │
│  ✅ M₂ EQU = √(M₂(1)³×n₂(1)×t₁ + ... + M₂(n)³×n₂(n)×tₙ) /│
│              (n₂(1)×t₁ + ... + n₂(n)×tₙ)                  │
│                                                             │
│  ✅ n₂ EQU = (n₂(1)×t₁ + n₂(2)×t₂ + ... + n₂(n)×tₙ) /    │
│              (t₁ + t₂ + ... + tₙ)                          │
│                                                             │
│  ✅ ED% = (ED × 100) / t                                   │
│                                                             │
│  🔥 FACTORS:                                               │
│                                                             │
│  ✅ Speed Factor (fₙ) - Based on input speed               │
│  ✅ Temperature Factor (fₜ) - Derating for >30°C           │
│  ✅ Cycle Factor (f₂) - Based on cycles/hour               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
