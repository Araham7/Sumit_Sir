## 📖 NOTATION REFERENCE

| Code | Meaning |
|:----:|:--------|
| **BMS** | Bonfiglioli Motion Solution (Integrated Motor + Gearbox) |
| **F** | Flanged Version (Flange Output) |
| **160/130/090** | Gearbox Size (Frame Size) |
| **M** | Medium Overspeed (Balanced performance) |
| **50** | Gear Ratio (50:1) |
| **145B/102A** | Motor Size |
| **400** | Voltage Supply (400 VAC, 3-Phase) |
| **LOW** | Reduced Backlash (≤5' for 1-stage, ≤7' for 2-stage) |
| **S1** | Continuous Duty |
| **UH1** | Food Grade Synthetic Lubricant |
| **PTC** | Thermal Protection (PTC Thermistor) |
| **ENB11** | Hiperface DSL Absolute Encoder (20+ bit) |
| **F1** | Additional Flywheel / Inertia |
| **F24** | 24V Electromagnetic Holding Brake |
| **AN** | Angled Turning Receptacles |
