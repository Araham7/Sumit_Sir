```

│  ✅ Back EMF Constant (Kₑ): Voltage/speed ratio             │
│  ✅ Torque Constant (Kₜ): Current/torque ratio              │
│                                                             │
│  ✅ Electric Time Constant: τₑₗ = Lₚₚ / Rₚₚ                 │
│  ✅ Rated Frequency: fₙ = p × nₙ / 60                       │
│  ✅ Rated Power: Pₙ = 2π × Mₙ × nₙ / 60                     │
│                                                             │
│  ⭐ Thermal equilibrium = Temperature rise ≤ 1K/30 min      │
│                                                             │
```
